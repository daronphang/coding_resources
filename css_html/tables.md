## Drawing Tables:
```
<div class="container">
  <table class="table-sm">    #sm/md/lg/xl
    <thead>
      <tr>
        <th> Header1 </th>
        <th> Header2 </th>
        <th> Header3 </th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td> row1 column1 </td>
        <td> row1 column2 </td>
        <td> row1 column3 </td>
      </tr>
      <tr>
        <td> row2 column1 </td>
        <td> row2 column2 </td>
        <td> row2 column3 </td>
      </tr>
    </tbody>
  </table>
</div>
```
