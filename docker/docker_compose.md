## Docker Compose:
Consists of YAML file. YAML file can be used with docker-compose for local docker automation or with docker directly in production with Swarm.

```
version: "3.9"  # optional since v1.27.0
services:
  web:
    build: .
    ports:
      - "5000:5000"
    volumes:
      - .:/code
      - logvolume01:/var/log
    links:
      - redis
  redis:
    image: redis
volumes:
  logvolume01: {}
```
